# Note Upgrade Instructional Video

## Audience

Supply chain analysts using Lyric Studio.

## Objective

Show how to upgrade a Note to a newer version.

## Video flow

1. Show the Note in the workspace.
2. Right-click **Optimize Network**.
3. Select **Upgrade Note**.
4. Show the current version in the Note version window.
5. Click **Change Version**.
6. Select a new version from the dropdown.
7. Highlight the schema differences.
8. Click **Upgrade Note**.

# Alternative Release Note Format

## Introduction to Bar Chart

As part of this release, a Bar chart has been added to the list of available chart types in Apps Studio. This feature gives users a visual way to compare values across categories and review multiple measures side by side within an interface page.

Users can add a Bar chart, select fields for the X-axis and Y-axis, and apply basic style changes such as chart title, series colors, and labels. This addition helps make category-based data easier to review and interpret.

## Impacted users

This feature impacts users who:

- build interface pages in Apps Studio
- configure charts for data visualization
- review category-based data in application pages

## What has changed?

Previously, the available chart options did not include a Bar Chart for category-based comparisons in the flow shown in the assignment material.

With this release, users can now:

- add a **Bar** chart from the **Add Chart** menu.
- enter a chart title using **Widget Title**.
- configure the chart using X-Axis and Y-Axis.
- customize the chart by changing series colors.
- turn on labels for the chart series.
- apply the chart to the interface page.

This makes it easier to compare values across groups such as regions and to view multiple measures within the same chart.

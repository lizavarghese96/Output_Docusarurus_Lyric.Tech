# Bar Chart

A bar chart shows values for different categories as rectangular bars. It helps you compare values across groups and quickly see differences in size.

In Apps Studio, you can add a bar chart to an interface page and configure it by selecting fields for the X-axis and Y-axis. You can also adjust the chart title and style settings.

## When to use a bar chart

Use a bar chart when you want to:

- compare values across categories
- view measures side by side for the same category
- highlight differences across groups such as regions

For example, a bar chart can help you compare production and demand across regions.

Perform the following steps to add a bar chart in the Apps Studio: 

1. Navigate to the required **Interface**.
![alt text](../static/img/Screen1.png)

2. Click on **Chart** widget in the widget tray.
3. Select **Bar** from the **Add Chart** menu.
![alt text](../static/img/Screen2.png)

4. Enter a name for the chart in the **Widget Title** field in the configuration panel displayed on the right side of the page.
![alt text](../static/img/Screen3.png)

5. Click on the **Add Column** drop-down menu from the **X Axis** menu under the **Data** tab.
![alt text](../static/img/Screen4.png)

6. Select the required column to use as the category axis and repeat the same as Step 6 to select the required columns from **Y Axis** menu.

***Note:** Users can select more than one column for both menus as required.*
![alt text](../static/img/Screen5.png)

After you add the X-axis and Y-axis fields, the chart preview is updated.

7. Scroll to the bottom page and click on **Customize Series** drop-down menu. 
8. Click on Pick Color for Production icon to customise the columns which is indicated as the bar color.
![alt text](../static/img/Screen6.png)

In the example shown, separate colors are applied to **Production** and **Demand**.

10. Click on the Series Settings icon to configure the label.
![alt text](../static/img/Screen7.png)

11. Toggle the **Show Labels** button.
12. Click on **Apply** button.

In the example shown, labels are displayed on the bars.

13. Review the bar chart details and click on **Apply** button.

**Result:** The bar chart is added to the **Interface** page.
![alt text](../static/img/Screen8.png)
